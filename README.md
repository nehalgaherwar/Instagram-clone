# Instagram Clone

A simple front-page demo of Instagram built using HTML, CSS, and JavaScript.

## Features

- Header with Instagram logo, search bar, and navigation icons
- Feed with sample posts including user avatars, images, like buttons, and comments
- Interactive like buttons that update the like count
- Responsive design

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## How to Run

1. Open `index.html` in a web browser
2. The demo will load with sample posts using online images

## Note

This is a static demo and does not include backend functionality like user authentication, posting, or real-time updates.
